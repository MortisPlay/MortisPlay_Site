## 🛠️ Технологический стек

### Frontend
* **HTML5:** Семантическая и доступная разметка (использование тегов `main`, `section`, `header`, `footer`, атрибутов `aria-label`).
* **CSS3 & Tailwind CSS:** Утилитарный CSS-фреймворк для мгновенного рендеринга стилей и адаптивной сетки (`grid`, `flex`).
* **JavaScript (ES6+):** Чистый асинхронный код без jQuery, использующий `async/await`, `fetch`, ООП-классы и паттерн `debounce` для оптимизации поиска.
* **Font Awesome v6.5.0:** Пакет векторных иконок высокого разрешения.
* **Google Fonts:** Премиальный читаемый шрифт *Inter*.

### API и Интеграции
* **YouTube Data API v3:** Автоматическое обновление архива видео и сбор метаданных канала.
* **Twitch Embed API:** Интеграция стриминг-контента с безопасной привязкой к хосту.
* **Google Analytics (gtag.js):** Отслеживание пользовательской активности и кликов по видео/клипам.

## 🎯 Инструкция для пользователей

### Для зрителей
1. Зайдите на сайт и выберите тему (день/ночь) с помощью кнопки-солнца в шапке.
2. Включайте VPN, если видеохостинг работает нестабильно (инструкция доступна по кнопке в разделе видео).
3. Следите за обновлениями в ленте «Новости».
4. Поддерживайте проект рублем через официальную кнопку **DonationAlerts**.

### Для разработчиков
1. Логика работы видео реализована через изолированный класс `VideoSystem` в `index.html`.
2. Конфигурация API находится на стыке тега `<script>` в объекте `YOUTUBE_CONFIG`.
3. Верстка полностью адаптивна и протестирована под экраны смартфонов, планшетов и 4K-мониторов.
